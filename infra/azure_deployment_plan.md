# Azure Deployment Plan

## Project
HuskyPark Predictor: AI-Driven Parking Spot Probability Dashboard for SCSU

## Goal
Prepare the HuskyPark Predictor system for deployment on Microsoft Azure.

## Components to Be Deployed
- React frontend
- FastAPI backend
- PostgreSQL relational database
- MongoDB NoSQL database

## Planned Cloud Setup
- The frontend will be hosted as a web application service.
- The backend API will be deployed using Azure App Service or Azure Container Apps.
- PostgreSQL will be hosted using Azure Database for PostgreSQL.
- MongoDB data will be hosted using a managed MongoDB-compatible cloud service.
- Application configuration and secrets will be stored using environment variables.

## DevOps Preparation
- GitHub will be used for source control and collaboration.
- Branching and pull requests will support controlled updates.
- Project folders are organized for backend, frontend, infrastructure, and documentation.

## Expected Outcome
The project will be prepared for version-controlled collaboration and future cloud deployment in Azure.
